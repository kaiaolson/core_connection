## Set up

### Required environment variables

```env
# Amazon S3 to store profile files
S3_KEY
S3_SECRET
S3_REGION
S3_BUCKET_NAME

# Gmail account for sending emails
email_user_name
email_password
```

### Administrator

Don't forget to create a user and toggle their `admin` field so they can approve user profiles.
